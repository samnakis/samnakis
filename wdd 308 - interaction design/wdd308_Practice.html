WDD308 Instructions

Create an html file and create a link to a javascript file.

Current Course

  1. Create the following variables with any values you want:
    • Course Number (with designator)
    • Course Name
    • Credit Hours
    • Required Class (true or false)
  2. Create another variable that will marry the course number with the course name with a colon in between. Ex. WDD308: Interaction Design
  3. Print out the following so that it always appears at the bottom of the HTML page:
    Ex. I am currently enrolled in WDD308: Interaction Design.

Course Load

  1. Create an array that will hold the number or credit hours you have taken for the past four semesters.
  2. Create a paragraph in your HTML page. Give it an id of "creditHrs."
  3. Add a horizontal line (hr) after the paragraph.
  4. Target that paragraph in your javascript document and create an ordered list with the following information:
    1. Credit Hours by semester: (print the entire array)
    2. Total Number of Credit Hours: (print total credit hours)
    3. Average per year: (print average number of credit hours per year)
    4. Average per semester: (print average number of credit hours per semester)

School Supplies

  1. You are buying a school supplies at the Butler Bookstore. Here are the list of items for sale:
    • Textbook: $75
    • Notebook: $5
    • Calculator: $30
    • Pen: $2
      • Pencil: $.75
    • USB Drive: $6
  2. You need to purchase 2 textbooks, 1 notebook, 1 calculator, 4 pens, 3 pencils and 1 USB drive.
  3. You get a student discount oif 10%.
  4. Create a div on your HTML document and give it an ID.
  5. Create an ordered list and list the following:
    1. SubTotal: (subtotal of all items before discount and tax)
    2. Discount Savings: (dollar amount of your discount
    3. Total: (calculate total - discount and then add tax to new total)
      NOTE: Tax rate is 0.08

Your Information

  1. Create an object that holds the following information about yourself:
    • First name
    • Last name
    • Age
    • Address
    • Hometown
    • State
    • FullAddress (this will be a method that will create your address that will look like this:
      123 Main Street
      Indianapolis, Indiana
  2. Create an h2 in your HTML document that reads: My Information.
  3. Create a paragraph that follows the h2 and give it an id.
  4. Add a horizontal line (hr) after the paragraph.
  5. Target this paragraph to display the following information:
    Sally Smith
    123 Main Street
    Indianapolis, Indiana

My Friends

  1. Create an h2 in your HTML document that reads: My Friends.
  2. Create two paragraphs that follows the h2 and give it each an id.
  3. Add a horizontal line (hr) after the paragraph.
  4. Create a constructor object called friends that will allow you to create multiple friend objects. You should include the following information:
    • First name
    • Last name
    • Age
    • Email
    • How you met
    • Fun fact
  5. Create two friends.
  6. Use the paragraphs to print out the following for each friend:
